# Gaming DVD Recommender System (UBCF)
# Business Objective
To build a recommender system that suggests top-selling gaming DVDs to customers to improve store sales.
# Project Description
This project uses User-Based Collaborative Filtering (UBCF) to recommend games based on user similarity.
# Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- User-Item Matrix Creation
- Cosine Similarity Calculation
- Recommendation Engine using UBCF
## Model Used
User-Based Collaborative Filtering (UBCF) with Cosine Similarity.
## How the Recommender System Works
- Finds similar users based on rating patterns
- Identifies top similar users
- Recommends DVDs liked by similar users
- Excludes already rated DVDs
# Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
# Outcome
The model recommends unseen DVDs to users based on similar user preferences, helping increase sales and customer satisfaction.
